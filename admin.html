Blossom like crocus 文章管理系统

发布步骤

  1. 填写下方表单，点击"生成文章文件"
  2. 复制生成的 JSON 内容
  3. 在 GitHub 仓库的 posts/ 文件夹下创建新文件
  4. 文件名格式：YYYY-MM-DD.json（如 2024-03-20.json）
  5. 粘贴内容，保存提交，博客首页自动更新

推荐 Unsplash 图片，右键复制图片地址

用空行分隔段落，支持多段文字
