设置 Token

配置 GitHub Token

用于一键推送到仓库（仅存储在本地浏览器）

Token 仅保存在浏览器本地存储，不会被上传到任何地方

获取方式：GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → 勾选 repo 权限

发布步骤

  1. 在"写文章"标签填写标题、日期、正文
  2. 点击"生成文章文件"，复制 JSON 内容或一键推送
  3. 如果手动上传：在 GitHub 仓库的 posts/ 文件夹创建 YYYY-MM-DD.json
  4. 博客首页会自动加载新文章，按日期排序

三种发布方式：

  • 一键推送：配置 Token 后点击"推送到 GitHub"
  • 下载文件：点击"下载 JSON"，手动上传到 posts/ 文件夹
  • 复制粘贴：复制代码框内容，在 GitHub 新建文件粘贴

推荐 Unsplash 图片，右键复制图片地址

用空行分隔段落，支持多段文字
