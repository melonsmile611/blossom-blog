🌸 管理设置

⚙️ 连接你的 GitHub 仓库

填写一次，网站就知道去哪里读你的 Obsidian 文章了

就是你在 GitHub 上建的那个仓库的名字

Obsidian 里存放文章的文件夹名，默认填 posts 就好

✓ 连接成功，文章已加载

✗ 连接失败，请检查用户名和仓库名

✍️ 每篇文章的格式

在 Obsidian 每篇文章最顶部加上这几行（叫 frontmatter），网站用这些信息生成卡片

--- title: 文章标题 tag: 圣经 excerpt: 一句简短的摘要，显示在卡片上 date: 2026-04-16 type: article --- 正文从这里开始写，支持完整 Markdown 格式...
1
tag 可填：圣经 AI极简 帕金森 儿童自闭症 YouTube 生活
2
type 可填：article（文章）youtube（视频，正文里放 YouTube 链接）quote（短句）
3
文章写好后，在 Obsidian 用 Obsidian Git 插件一键同步到 GitHub，网站约 1 分钟后自动更新

🔌 Obsidian Git 插件设置

1
Obsidian → 设置 → 第三方插件 → 浏览 → 搜索 Obsidian Git → 安装并启用
2
插件设置里开启 Auto pull on startup 和 Auto push，推送间隔可设为 0（手动）
3
写完文章后：按 Ctrl + P → 搜索 Obsidian Git: Commit and push → 回车，完成发布
4
也可以在左侧工具栏点 Git 图标，点「Commit and push all changes」
